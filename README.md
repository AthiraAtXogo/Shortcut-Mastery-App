# Shortcut Mastery App

A gamified keyboard shortcut learning application. Learn shortcuts for Windows, VS Code, Chrome, AI tools, and more — without getting bored.

## Features

- **Game Modes**: Flash Round, Speed Run, Boss Battle, Survival Mode
- **Gamification**: XP, levels, streaks, achievements, leaderboards
- **Smart Learning**: Spaced repetition targets your weak spots
- **Offline Support**: Works without internet (PWA)
- **Multiple Apps**: Windows, macOS, VS Code, Chrome, Claude, ChatGPT, and more

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- **Framework**: Nuxt 4 + Vue 3
- **UI**: Nuxt UI + Tailwind CSS
- **Storage**: IndexedDB (offline-first)
- **Audio**: Howler.js
