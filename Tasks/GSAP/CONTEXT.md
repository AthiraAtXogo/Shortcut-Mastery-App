# GSAP - Feature Context

## Purpose

Create reusable GSAP animation utilities and effects for UI interactions, text effects, and feedback animations.

## Dependencies

- Core/03-gsap-setup (GSAP installed)

## Tasks Overview

| Task | Description | Priority |
|------|-------------|----------|
| 01-text-scramble | Matrix-style text reveal | P0 |
| 02-counter-roll | Slot machine number animation | P0 |
| 03-elastic-bounce | Satisfying button feedback | P0 |
| 04-stagger-cascade | List item reveal | P1 |
| 05-flip-cards | 3D card flip for Memory Match | P1 |
| 06-shake-flash | Wrong answer feedback | P0 |
| 07-confetti-burst | Achievement celebration | P1 |
| 08-draw-on-lines | Progress bar animation | P1 |

## Animation Principles

- All animations respect reduced motion preference
- Keep durations short (0.2-0.6s for most)
- Use appropriate easing for each context
- Provide immediate feedback
