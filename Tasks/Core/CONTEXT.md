# Core - Feature Context

## Purpose

Set up the foundational infrastructure for the Shortcut Mastery App including Nuxt 4 configuration, PWA support, animation libraries, and data layer.

## Dependencies

None - this is the first feature to implement.

## Tasks Overview

| Task | Description | Priority |
|------|-------------|----------|
| 01-nuxt-setup | Initialize Nuxt 4, TypeScript, Tailwind | P0 |
| 02-pwa-offline | Configure PWA for offline support | P0 |
| 03-gsap-setup | Install and configure GSAP | P0 |
| 04-threejs-tresjs | Set up Three.js with TresJS Vue wrapper | P0 |
| 05-howler-audio | Audio system for sound effects | P1 |
| 06-dexie-database | IndexedDB setup with Dexie.js | P0 |
| 07-state-management | Pinia stores structure | P0 |

## Technical Notes

- Use `pnpm` as package manager
- Target Node 20+
- Enable strict TypeScript
- Configure Tailwind with custom design tokens
