# Design - Feature Context

## Purpose

Create the premium visual design system with holographic cards, plasma borders, magnetic cursor effects, and dark gaming aesthetic.

## Dependencies

- Core/01-nuxt-setup (Tailwind CSS)
- Core/03-gsap-setup (animations)

## Tasks Overview

| Task | Description | Priority |
|------|-------------|----------|
| 01-design-tokens | Colors, spacing, typography variables | P0 |
| 02-holographic-cards | Rainbow shimmer, 3D tilt effect | P0 |
| 03-plasma-borders | Animated gradient borders | P1 |
| 04-magnetic-cursor | Elements pull toward mouse | P1 |
| 05-button-system | Interactive button components | P0 |
| 06-typography | Font system, text styles | P0 |
| 07-dark-theme | Dark gaming aesthetic | P0 |

## Design Principles

- Dark, immersive gaming aesthetic
- Neon accents with electric blue primary
- Satisfying micro-interactions
- Premium feel without being overwhelming
- Reduced motion support always
